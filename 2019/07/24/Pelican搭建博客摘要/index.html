日常使用

如果本地没有，先clone下来

  2. cd m-lh.github.io
  3. 切换到source分支，不过默认就是source分支。git branch source

步骤如下:

  1. 在content目录添加文章
  2. inv livereload
  3. git add 内容
  4. git commit
  5. git push –set-upstream origin source
  6. inv gh-pages
